# Create a new ChartPart containing `chart_type` data from `chart_data`

Create a new ChartPart containing `chart_type` data from `chart_data`

## Usage

``` r
ChartPart_new(chart_type, chart_data, package)
```

## Arguments

- chart_type:

  An integer from XL_CHART_TYPE.

- chart_data:

  A CategoryChartData, XyChartData, or BubbleChartData.

- package:

  The OPC package.

## Value

A ChartPart.
