# Mouse action settings on a shape

Provides access to click and hover action properties of a shape,
including hyperlink URL and action type. Access via
`shape$click_action`.

## Methods

### Public methods

- [`ActionSetting$new()`](#method-ActionSetting-new)

- [`ActionSetting$clone()`](#method-ActionSetting-clone)

------------------------------------------------------------------------

### Method `new()`

#### Usage

    ActionSetting$new(cNvPr, parent, hover = FALSE)

------------------------------------------------------------------------

### Method `clone()`

The objects of this class are cloneable with this method.

#### Usage

    ActionSetting$clone(deep = FALSE)

#### Arguments

- `deep`:

  Whether to make a deep clone.
