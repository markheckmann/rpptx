# Define a ZeroOrOne child element specification

Define a ZeroOrOne child element specification

## Usage

``` r
zero_or_one(nsptagname, successors = character(0))
```

## Arguments

- nsptagname:

  Namespace-prefixed tag, e.g. `"p:sldIdLst"`.

- successors:

  Character vector of successor tag names for ordering.

## Value

A list describing the child element spec.
