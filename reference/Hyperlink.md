# Hyperlink on a text run

Provides read/write access to the URL of a hyperlink on an `a:r`
element. Access via `run$hyperlink`.

## Methods

### Public methods

- [`Hyperlink$new()`](#method-Hyperlink-new)

- [`Hyperlink$clone()`](#method-Hyperlink-clone)

------------------------------------------------------------------------

### Method `new()`

#### Usage

    Hyperlink$new(rPr, slide_part)

------------------------------------------------------------------------

### Method `clone()`

The objects of this class are cloneable with this method.

#### Usage

    Hyperlink$clone(deep = FALSE)

#### Arguments

- `deep`:

  Whether to make a deep clone.
