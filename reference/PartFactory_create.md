# Construct the appropriate Part subclass for a given content type

Construct the appropriate Part subclass for a given content type

## Usage

``` r
PartFactory_create(partname, content_type, package, blob)
```

## Arguments

- partname:

  A PackURI.

- content_type:

  Content type string.

- package:

  The package.

- blob:

  Raw bytes.

## Value

A Part instance.
