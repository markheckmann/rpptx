# Signal an rpptx error

Signal an rpptx error

## Usage

``` r
rpptx_error(message, class = NULL, call = NULL, ...)
```

## Arguments

- message:

  Error message.

- class:

  Character vector of additional subclasses.

- call:

  The call to include in the error.

- ...:

  Additional fields for the condition object.

## Value

No return value; raises a condition.
