# Hyperlink on a shape

Provides read/write access to a URL hyperlink on a shape. Access via
`shape$click_action$hyperlink`.

## Methods

### Public methods

- [`ShapeHyperlink$new()`](#method-ShapeHyperlink-new)

- [`ShapeHyperlink$clone()`](#method-ShapeHyperlink-clone)

------------------------------------------------------------------------

### Method `new()`

#### Usage

    ShapeHyperlink$new(cNvPr, slide_part, hover = FALSE)

------------------------------------------------------------------------

### Method `clone()`

The objects of this class are cloneable with this method.

#### Usage

    ShapeHyperlink$clone(deep = FALSE)

#### Arguments

- `deep`:

  Whether to make a deep clone.
