# rpptx: Create, Read, and Update PowerPoint Presentations

Create, read, and update PowerPoint (.pptx) files. An R port of the
Python 'python-pptx' library. Provides an R6-based object model for
manipulating presentations, slides, shapes, text, charts, tables, and
formatting properties. Supports autoshapes, text boxes, pictures,
tables, charts (column, bar, line, pie, scatter, bubble, radar,
doughnut), fills, lines, colors, freeform custom-geometry shapes, and
placeholder inheritance.
