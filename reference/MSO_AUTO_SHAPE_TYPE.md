# MSO_AUTO_SHAPE_TYPE — preset geometry names for autoshapes

Each entry is a list with `value` (MSO integer constant) and `prst`
(OOXML `prst` attribute string). Pass a member to
`SlideShapes$add_shape()`.

## Usage

``` r
MSO_AUTO_SHAPE_TYPE

MSO_SHAPE
```

## Format

An object of class `list` of length 107.

An object of class `list` of length 107.

## Details

Alias: `MSO_SHAPE`
