# Create an RGB color value

Represents a color as (red, green, blue) components in 0–255 range.

## Usage

``` r
RGBColor(r, g, b)
```

## Arguments

- r:

  Integer 0–255 red component.

- g:

  Integer 0–255 green component.

- b:

  Integer 0–255 blue component.

## Value

An `RGBColor` object.
