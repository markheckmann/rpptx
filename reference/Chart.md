# Chart domain object

Provides access to chart properties, title, legend, plots, series, and
axes. Access via `graphic_frame$chart`.

## Methods

### Public methods

- [`Chart$new()`](#method-Chart-new)

- [`Chart$clone()`](#method-Chart-clone)

------------------------------------------------------------------------

### Method `new()`

#### Usage

    Chart$new(chartSpace, chart_part)

------------------------------------------------------------------------

### Method `clone()`

The objects of this class are cloneable with this method.

#### Usage

    Chart$clone(deep = FALSE)

#### Arguments

- `deep`:

  Whether to make a deep clone.
