# Create a Relationship from a CT_Relationship XML element

Create a Relationship from a CT_Relationship XML element

## Usage

``` r
Relationship_from_xml(base_uri, rel, parts)
```

## Arguments

- base_uri:

  Base URI string.

- rel:

  A CT_Relationship element.

- parts:

  Named list mapping partname -\> Part.

## Value

A Relationship object.
