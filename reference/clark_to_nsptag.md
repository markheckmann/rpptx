# Convert a Clark-notation tag to a namespace-prefixed tag

Convert a Clark-notation tag to a namespace-prefixed tag

## Usage

``` r
clark_to_nsptag(clark_name)
```

## Arguments

- clark_name:

  A Clark-notation tag, e.g.
  `"{http://schemas.openxmlformats.org/presentationml/2006/main}cSld"`.

## Value

A namespace-prefixed tag string, e.g. `"p:cSld"`.
